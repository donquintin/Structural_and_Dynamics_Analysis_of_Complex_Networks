# Structural and Dynamics Analysis of Complex Networks

Final project for the Complex Networks course. It has two independent parts:

- **`StructuralAnalysis/`** — structural characterization of a real-world network:
  the network of Mexican municipalities linked by shared cartel presence and
  geographic proximity (degree distribution, correlations, clustering,
  communities, and comparison against a configuration-model null model).
- **`SISDynamics/`** — SIS epidemic dynamics on synthetic scale-free networks,
  characterizing the non-equilibrium phase transition with the lifespan method
  (Mata et al., PRE **91**, 052117).

## Repository layout

```
StructuralAnalysis/
    code_mex.ipynb          # structural analysis notebook
    data/                   # network + geometry data (see "Data" below)
    iter_00_cpv2010/        # INEGI 2010 census (population) — not in repo
SISDynamics/
    functions_2.py          # Gillespie SIS engine + network generation
    lifespan_analysis.py    # peak location + finite-size-scaling fits
    generate_networks.py    # builds the synthetic networks
    run_sweep.py            # one lifespan sweep per (gamma, N)
    rerun_all.py            # runs all sweeps sequentially
    make_double_panels.py   # builds the figures
    analisis_results_2.ipynb# analysis notebook
```

## Data

The large data files are **not** included in the repository (they exceed
GitHub's size limits). Download them and place them as indicated:

1. **Cartel presence network** — `StructuralAnalysis/data/municipalities_and_carteles.csv`
   Source: *Mexico Drug Traffic Activities* dataset by M. Coscia,
   <https://www.michelecoscia.com/?page_id=1032>
   (associated paper: Coscia & Rios, CIKM 2012, DOI `10.1145/2396761.2398446`).

2. **Municipal boundaries (geometry)** — `StructuralAnalysis/data/municipalities_full.geojson`
   Source: M. Mora, *Mexico GeoJSON — States and Municipalities*, figshare,
   DOI `10.6084/m9.figshare.31236322` (<https://doi.org/10.6084/m9.figshare.31236322>).
   Derived from CONABIO shapefiles. Download `municipalities_full.geojson`.

3. **Municipal population (census)** — `StructuralAnalysis/iter_00_cpv2010/`
   Source: INEGI, *Censo de Población y Vivienda 2010*, "Principales resultados
   por localidad (ITER)", <https://www.inegi.org.mx/programas/ccpv/2010/>.
   Download the national ITER dataset and unzip it so that
   `iter_00_cpv2010/conjunto_de_datos/iter_00_cpv2010.csv` exists.

For the SIS part, the synthetic networks and result files are **generated by the
code** (see below); they are not stored in the repository.

## Requirements

```
pip install numpy scipy matplotlib networkx numba pandas geopandas
```
`numba` is optional but strongly recommended for the SIS engine (~50–100x faster).
`pandas`/`geopandas` are only needed for the structural-analysis notebook.

## How to run

**Structural analysis:** open and run `StructuralAnalysis/code_mex.ipynb`
(requires the three datasets above in place).

**SIS dynamics** (from `SISDynamics/`):
```
# 1) generate the synthetic networks (first time use --force)
python generate_networks.py --force

# 2) run the lifespan sweeps
python rerun_all.py --networks networks --results results --force
#   or one job per (gamma, N):
python run_sweep.py 3.5 1000000 --results results --force

# 3) analysis and figures
python make_double_panels.py        # or run analisis_results_2.ipynb
```
Adjust `RESULTS_DIR` / `FIG_DIR` inside `make_double_panels.py` and the notebook
to match the results folder used in step 2.
